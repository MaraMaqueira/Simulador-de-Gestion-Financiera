  Gestión de Ingresos y Gastos		
	
Este proyecto es una aplicación web para la gestión de ingresos y gastos personales. Permite a los usuarios registrar y visualizar sus ingresos y gastos, así como crear categorías personalizadas para cada uno. Además, incluye funciones para filtrar y buscar transacciones, y manejar presupuestos y ahorros mensuales.

  Características
	
Inicio de Sesión y Registro: Los usuarios pueden registrarse y acceder a su cuenta.
Gestión de Ingresos: Opción para agregar, ver, buscar y filtrar ingresos por nombre, categoría y fecha.
Gestión de Gastos: Funcionalidades similares para los gastos, incluyendo la creación de categorías.
Ahorros y Presupuestos: Los usuarios pueden ingresar su presupuesto mensual y visualizar sus ahorros totales.
Interfaz de Usuario Intuitiva: Diseño simple y fácil de usar que facilita la navegación.
